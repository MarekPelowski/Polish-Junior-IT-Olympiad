Zadanie: Kalkulator Operacji na Liczbach
Napisz program kalkulatora, który obsługuje proste operacje na dwóch liczbach. Program powinien umożliwiać dodawanie, odejmowanie, mnożenie i dzielenie.

Napisz funkcję Dodaj, która przyjmuje dwie liczby i zwraca ich sumę.
Napisz funkcję Odejmij, która przyjmuje dwie liczby i zwraca różnicę pierwszej od drugiej.
Napisz funkcję Pomnóż, która przyjmuje dwie liczby i zwraca ich iloczyn.
Napisz funkcję Podziel, która przyjmuje dwie liczby i zwraca iloraz pierwszej przez drugą. Pamiętaj o obsłudze przypadku dzielenia przez zero.
Następnie napisz program główny, który pyta użytkownika, jaką operację chce wykonać (dodawanie, odejmowanie, mnożenie, dzielenie) i wczytuje od niego dwie liczby. W zależności od wybranej operacji, program powinien użyć odpowiedniej funkcji i wyświetlić wynik.

Wejście:

Pierwsza linia zawiera wybór operacji: "Dodawanie", "Odejmowanie", "Mnożenie" lub "Dzielenie".
Kolejne dwie linie zawierają dwie liczby rzeczywiste (liczby zmiennoprzecinkowe) do wykonania operacji.
Wyjście:

Wypisz wynik operacji w formie liczby zmiennoprzecinkowej zaokrąglonej do dwóch miejsc po przecinku.
Przykład:

makefile
Copy code
